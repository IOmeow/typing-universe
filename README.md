# Typing Universe

Turn your keyboard into a **spinning letter universe**!  
Every key you type spawns a **rotating 2D letter** at a **random position** on your screen.  
Letters **slow down and fade out**, creating a dynamic generative art overlay.

![Demo GIF](demo.gif)

## Features

- Letters appear at **random positions** on your screen  
- Each letter **rotates**, gradually slows down, and fades out  
- **Mouse attracts letters** for interactive control  
- **Control panel** to adjust parameters (speed, rotation, fade, etc.)  
- Runs on **Windows / macOS / Linux** using **Tauri + p5.js**  
- Lightweight, fun, and interactive overlay  

## How to Use

1. Download the latest release for your platform  
2. Run the executable  
3. Start typing anywhere and see letters appear on screen  
4. **Right-click the tray icon** to:
   - Toggle **full screen**  
   - Show / hide the overlay
   - Open the **Control Panel** to adjust size, gravity, or other parameters
   - Exit  

## Notes

- This overlay is **always on top**, so be careful when typing in other apps  
- Designed for fun, creative coding experiments  
- Works best with continuous typing and mouse interaction
